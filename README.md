# Full Stack Developer Profile (React + TypeScript + Express)

This repository contains a minimal full-stack developer profile application:

- `client` — React + Vite Single Page Application (TypeScript)
- `server` — Express API (TypeScript) serving profile JSON

Quick start (Windows):

1. Install root helper and both packages:

```powershell
npm install
npm run install-all
```

2. Run both dev servers:

```powershell
npm run dev
```

Open the client at http://localhost:5173 (Vite). API requests to `/api` are proxied to the server at port 4000.

You can build production bundles separately in `client` and `server`.
